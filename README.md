# Jenya Mobile React Starter

Mobile-first React website structure built with Vite.

## Structure

- `src/components` reusable UI pieces
- `src/layouts` app shells and wrappers
- `src/pages` page-level screens
- `src/styles` global and shared styles
- `src/assets` images, icons, and media
- `public` static files

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:5173`
